# הוראות הרצה מהירות

## התקנה
```bash
cd recipe-app
npm install
```

## הרצה
```bash
npm start
```

## פתיחת הדפדפן
פתח את הדפדפן בכתובת: http://localhost:4200

## התחברות
- שם משתמש: `admin`
- סיסמה: `123456`

או:
- שם משתמש: `user`
- סיסמה: `password`

## תכונות לבדיקה

### 1. מסך התחברות
- נסה להתחבר עם הפרטים למעלה
- נסה להתחבר עם פרטים שגויים (תקבל הודעת שגיאה)

### 2. רשימת מתכונים
- חפש מתכון לפי שם או חומר
- לחץ על "הוסף מתכון" כדי להוסיף מתכון חדש
- לחץ על מתכון כדי לראות את הפרטים
- העבר עכבר על מתכון ולחץ על כפתור המחיקה

### 3. פרטי מתכון
- לחץ על "השמע" כדי להאזין להוראות
- לחץ על "השהה" כדי לעצור זמנית
- לחץ על "המשך" כדי להמשיך
- לחץ על "עצור" כדי לעצור לגמרי

### 4. הגדרות
- לחץ על כפתור ההגדרות (גלגל השיניים) בראש הדף
- שנה את זמן ההמתנה בין שורות
- הפעל/כבה מסך כהה
- לחץ על "שמור הגדרות"

### 5. התנתקות
- לחץ על כפתור ההתנתקות (חץ יציאה) בראש הדף

## בעיות נפוצות

### הקראה קולית לא עובדת
- ודא שאתה משתמש בדפדפן Chrome, Edge או Safari
- ודא שהדפדפן תומך בעברית
- בדוק שהקול לא מושתק

### האתר לא נטען
- ודא ש-Node.js מותקן
- הרץ `npm install` שוב
- נסה למחוק את תיקיית `node_modules` ולהתקין מחדש

## מידע נוסף
- כל הנתונים נשמרים ב-Local Storage של הדפדפן
- אם תנקה את ה-Cache, הנתונים יאופסו
- המתכונים וההגדרות נשמרים אוטומטית
