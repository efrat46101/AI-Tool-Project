# ספר המתכונים - אתר מתכונים עם הקראה קולית

## תיאור הפרויקט
אתר מתכונים אינטראקטיבי עם פונקציית הקראה קולית של הוראות ההכנה. האתר מאפשר למשתמשים לנהל מתכונים, לחפש ביניהם, ולהאזין להוראות ההכנה תוך כדי בישול.

## תכונות עיקריות

### ✅ מסך התחברות (Login)
- התחברות עם שם משתמש וסיסמה
- אימות משתמשים
- הגנה על דפים פנימיים

### ✅ מסך רשימת מתכונים (Recipe List)
- הצגת כל המתכונים בכרטיסים
- חיפוש מתכונים לפי שם או חומרים
- הוספת מתכון חדש
- מחיקת מתכונים
- ניווט להגדרות והתנתקות

### ✅ מסך פרטי מתכון (Recipe Detail)
- הצגת חומרים והוראות הכנה
- כפתורי הקראה קולית:
  - השמע - התחלת הקראה
  - השהה - עצירה זמנית
  - המשך - המשך הקראה
  - עצור - עצירה מוחלטת
- זמן המתנה מותאם אישית בין שורות

### ✅ מסך הגדרות (Settings)
- שינוי זמן המתנה בין שורות (0.5-5 שניות)
- מעבר בין מסך בהיר לכהה
- שמירת הגדרות אישיות

### ✅ אחסון מקומי (Local Storage)
- שמירת משתמשים
- שמירת מתכונים
- שמירת הגדרות אישיות
- שמירת מצב התחברות

## טכנולוגיות

- **Angular 20** - Framework
- **TypeScript** - שפת תכנות
- **SCSS** - עיצוב
- **Web Speech API** - הקראה קולית
- **Local Storage** - אחסון נתונים

## התקנה והרצה

### דרישות מקדימות
- Node.js (גרסה 18 ומעלה)
- npm או yarn

### התקנת תלויות
```bash
npm install
```

### הרצת שרת פיתוח
```bash
npm start
```
או
```bash
ng serve
```

האתר יהיה זמין בכתובת: `http://localhost:4200/`

### בניית הפרויקט לפרסום
```bash
npm run build
```

הקבצים יישמרו בתיקייה `dist/`

## מבנה הפרויקט

```
src/
├── app/
│   ├── components/
│   │   ├── login/           # מסך התחברות
│   │   ├── recipe-list/     # רשימת מתכונים
│   │   ├── recipe-detail/   # פרטי מתכון
│   │   └── settings/        # הגדרות
│   ├── guards/
│   │   └── auth-guard.ts    # הגנה על דפים
│   ├── models/
│   │   ├── recipe.interface.ts
│   │   └── user.interface.ts
│   ├── services/
│   │   └── data.ts          # ניהול נתונים והקראה קולית
│   └── app.routes.ts        # ניתוב
└── styles.scss              # סגנונות גלובליים
```

## פיצ'רים מתקדמים

### מנוע הקראה קולית
- תמיכה בעברית (he-IL)
- זמן המתנה מותאם אישית בין שורות
- כפתורי שליטה: השמע, השהה, המשך, עצור
- שמירת מצב ההקראה

### עיצוב רספונסיבי
- תמיכה מלאה במובייל
- אנימציות חלקות
- מעברים ויזואליים
- מסך כהה/בהיר

### ניהול מתכונים
- הוספה, עריכה ומחיקה
- חיפוש מתקדם
- תצוגת כרטיסים אטרקטיבית
